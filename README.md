### 🚀 Featured Projects

#### 🔍 Financial Fraud Detection — Deep Learning
> `TensorFlow` `Keras` `SMOTE` `XGBoost` `LightGBM` `scikit-learn`

Designed and trained a deep Artificial Neural Network (ANN) to classify fraudulent payment transactions in real time. Applied **SMOTE + MinMaxScaler** to handle extreme class imbalance and implemented **precision-recall threshold optimisation** to maximise F1 score beyond the default 0.5 cutoff. Benchmarked against Logistic Regression, XGBoost, and LightGBM using RandomizedSearchCV within sklearn Pipelines. Engineered a production-ready flow with EarlyStopping callbacks, stratified K-Fold cross-validation, and joblib model serialisation for deployment readiness.

---

#### ✈️ Customer Booking Conversion Prediction — British Airways (Forage)
> `Random Forest` `XGBoost` `SMOTE` `GridSearchCV` `Feature Engineering`

Binary classification pipeline trained on **50,000 customer records** to predict flight booking completion, achieving a recall of 0.62 on a heavily imbalanced dataset (15% positive class). Engineered **8 domain-specific features** including booking lead category, group size, long-haul flag, and extras score. Identified **booking origin (34% importance)** and **trip duration (20%)** as top predictors. Delivered a stakeholder PowerPoint deck recommending a targeted conversion strategy projected to deliver a **5–10% conversion lift** and **15–20% marketing cost reduction**.

---

#### 🏦 Bank Customer Churn Prediction — Lloyds Bank Group (Self-Initiated)
> `Logistic Regression` `Random Forest` `XGBoost` `SVM` `StratifiedKFold`

End-to-end churn prediction pipeline on **10,000+ customer records**, comparing four classifiers to identify the highest performer. Engineered **4 new financial behaviour features** (balance-per-product ratio, salary-balance dependency, age group, high-value flag). Addressed class imbalance via SMOTE and applied 5-fold StratifiedKFold cross-validation. Tuned Random Forest across **72 combinations** and XGBoost across **24 combinations** via GridSearchCV — findings directly applicable to retention campaign targeting for high-risk, high-value customers.

---

#### ⚙️ Production Churn Prediction System — MLOps Pipeline
> `FastAPI` `Docker` `MLflow` `DVC` `Prefect` `GitHub Actions` `scikit-learn`

Architected and deployed a **production-grade MLOps pipeline** covering the full model lifecycle: data versioning (DVC + DagsHub), experiment tracking (MLflow), orchestration (Prefect), and a containerised REST API serving real-time predictions. Published a Docker image (`emmanuely2000/bank-churn-api`) achieving **84% accuracy** and **63.1% weighted F1**. Implemented full CI/CD via GitHub Actions — automated tests on every commit, reducing manual deployment steps to zero. Delivered a browser-based HTML/JS frontend for non-technical stakeholders. Roadmap includes Evidently drift detection, scheduled Prefect retraining, SHAP explainability, and cloud deployment on GCP Cloud Run / AWS ECS.

---

#### 📊 Lounge Demand & Revenue Modelling — British Airways (Forage)
> `Python` `Matplotlib` `Seaborn` `Excel` `Statistical Modelling`

Analysed a **50,000+ flight schedule dataset** to model passenger lounge eligibility across 3 tiers, identifying peak demand windows of ~1,280 users/hour at 07:00–08:00. Built a revenue and cost model projecting **£262.8M annual profit** and recommended Concorde Room expansion with a projected **2-month payback** on a £7–10M investment. Engineered a multi-sheet Excel reporting workbook and a 6-panel Matplotlib/Seaborn dashboard. Model predictions landed within **3% of actual BA data** (long-haul predicted 41% vs actual 43.9%).
